# Modern Todo App

一个使用 React 和 TypeScript 构建的现代待办事项 Web 应用，具有响应式设计和现代 UI。

## 功能特性

- **任务管理**: 添加、编辑、删除和标记任务完成
- **进度追踪**: 实时显示任务完成进度
- **响应式设计**: 完美适配各种屏幕尺寸
- **现代 UI**: 使用 TailwindCSS 和 shadcn/ui 构建的精美界面
- **流畅动画**: 提供良好的用户交互体验

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

应用将在 `http://localhost:3000` 运行。

### 生产构建

```bash
pnpm build
```

### 预览构建结果

```bash
pnpm preview
```

## 使用说明

### 添加任务

1. 在输入框中输入任务内容
2. 点击"添加"按钮或按 Enter 键
3. 新任务将出现在列表顶部

### 完成任务

- 点击任务左侧的圆形图标标记任务完成
- 已完成的任务会显示为灰色并带有删除线

### 删除任务

- 点击任务右侧的垃圾桶图标删除任务

### 查看进度

- 页面顶部显示已完成/总任务数
- 进度条实时更新完成百分比
- 页面底部统计面板显示详细数据

## 设计特点

- **现代渐变背景**: 蓝色到靛蓝色的渐变背景
- **卡片式布局**: 使用 Card 组件组织内容
- **流畅过渡**: 所有交互都有平滑的动画效果
- **响应式设计**: 在移动设备、平板和桌面上都能完美显示
- **无障碍设计**: 支持键盘导航和屏幕阅读器
